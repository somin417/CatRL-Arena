# Assets

Place the GitHub Octocat image here:

```
assets/github_cat.png
```

Used by the **cat outline layout** in the interactive demo (`ui_duel.py`). The image is sampled into a 20×20 outline grid (white/gray border bricks only).

Headless training defaults to a rectangular brick grid and does not require this file.

**Attribution:** The Octocat character belongs to GitHub.
